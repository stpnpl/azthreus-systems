# Azthreus Systems — Public Website

Static website for Azthreus Systems. GitHub Pages compatible — no build step required.

## Pages

| File | Page |
|------|------|
| `index.html` | Home / Landing |
| `products.html` | Products (DTN Suite, QRC Core, RhAOS, Humanity Archive Platform) |
| `services.html` | Services (Architecture Consulting, V&V, Deep Space AI) |
| `about.html` | About (Mission, Vision, Values, Org Structure) |
| `contact.html` | Contact Form & Spokesperson Directory |

## Files

- `style.css` — All styles (inline only, no external dependencies)
- `nav.js` — Shared navbar & footer injected via JS
- `logo-v2.png` — Company logo
- `_config.yml` — GitHub Pages configuration

## Deployment

Push the contents of this folder to a GitHub repository's `main` branch and enable GitHub Pages (Settings → Pages → Source: `main`, folder: `/root`).
